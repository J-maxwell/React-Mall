
## 技术栈：

react + react-router + redux + immutable + less + ES6/7 + webpack + fetch + ant design


## 运行项目

```

 npm install
  
 npm run dev (正常编译模式) 或 npm run hot (热替换编译模式) 二选其一

 访问 http://localhost:8088

```
